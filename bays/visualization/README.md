# Visualization Bay

**Purpose:** Turn your data into charts.

A well-chosen chart communicates more than a table of numbers. This bay covers the most common chart types for CSV data exploration and reporting.

---

## What's in the notebook

- Histogram — distribution of a numeric column
- Bar chart — comparing categories
- Line chart — trends over time or sequence
- Scatter plot — relationship between two numeric columns
- Box plot — distribution and outliers by group
- Correlation heatmap — relationships across all numeric columns

---

## Choosing the Right Chart

| You want to show... | Use... |
|---|---|
| Distribution of one numeric column | Histogram |
| Comparison across categories | Bar chart |
| Change over time | Line chart |
| Relationship between two numeric columns | Scatter plot |
| Spread and outliers | Box plot |
| Relationships across many columns | Correlation heatmap |

---

## Libraries Used

- **Matplotlib** — base charting library, fine-grained control
- **Seaborn** — higher-level charts built on Matplotlib, better defaults for statistical plots

Both are installed by default in this repo.

---

## When to use this bay

- After analysis, to communicate findings
- When exploring distributions and relationships in new data (pair with the Inspection and Analysis bays)
- When preparing charts for reports or stakeholder presentations
